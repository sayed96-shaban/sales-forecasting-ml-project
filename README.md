# sales-forecasting-ml-project
Sales forecasting project using historical data. It analyzes sales trends and predicts future sales using time-series techniques like ARIMA.
It includes full time-series analysis, model building, forecasting, and visualization.

Project Features

 Data cleaning & preprocessing
 
 Trend visualization (matplotlib)
 
 ARIMA model (manual parameters)

Forecast for upcoming months

 Business-level insights



 Tech Stack

Python

Pandas

Matplotlib

Statsmodels (ARIMA)

pmdarima (Auto ARIMA)

Prophet
